Lewis-Clark State College iOS App

I developed this app with a team of other students for both my Software Engineering and as part of my Capstone Design class.

The Lewis-Clark State College iOS app provides students access to all the resources they need at the college. Through the app, students can access their student email, WarriorOne card, athletics livestream broadcasts, the KLCZ radio, and the college social media accounts. The app also features an event calendar that shows all the upcoming events on campus. Additionally, the students can add these events to their phone's calendar if they so desire.

I mainly developed the front and back ends of the campus event calendar using Objective-C. For my Capstone Design class, I also had to update the app to meet new requirements and to get it working with iOS 9.

Lurk Client/Server Project

This was my final project for my Networking class. It features the client and server implementations of a protocol my professor wrote for playing a text-based MUD.

The client is a Windows destop application that features a graphic user-interface. I used C# and Windows Presentation Foundation to write the client. The client is written to play on any server that implements the Lurk protocol.

The server is written in Python. It implements the portions of the protocol that the client uses, and also provides the implementation for the game environment itself. The environment on my server exists in the Star Trek universe and features a Federation ship under attack by an alien race similar to the Borg.

Lorem Ipsum

This is a project for my Artificial Intelligence class. It is a command-line Java application that colors maps according to the Four Color Theorem.

The application inputs a text file describing the borders of regions in a geographical map, and colors the regions so that no two bordering regions share the same color. It performs this task with the minimum number of different colors needed for any given map.

This is the first major Java project that I have worked on. On the left you can see a colored version of the Pacific Northwest. The circles represent the geographical regions, and the lines between each correspond to a border between those regions.
